Tables of Unicode Properties

Unicode Properties

These are Unicode properties recognized by Brasswork Regex. They are taken from http://www.unicode.org/Public/7.0.0/ucd/PropList.txt. Either the code or the name is accepted.

Code Name Description
Alpha Alphabetic Alphabetic characters. This is a superset of Letter.
Any All characters.
Assigned Code points with assigned characters; the complement of gc=Cn.
Lc CasedLetter Cased characters. This set includes Upper, Lower, and TitlecaseLetter.
Cntrl Control characters. Synonymous with gc=Cc.
DI Default Ignorable Code Point Code points that should be ignored by default in rendering unless explicitly supported.
Digit Decimal digits. Synonymous with gc=Nd. Also represented by \d.
Graph Characters with a visible representation.
Lower Lowercase Lowercase characters. This is a superset of LowercaseLetter.
NChar Noncharacter Code Point Code points that will never be assigned a Unicode character.
Punct Punctuation. Synonymous with gc=P.
Upper Uppercase Uppercase characters. This is a superset of UppercaseLetter.
Space Whitespace Whitespace characters. Also represented by \s.
Word Word characters. Also represented by \w.
